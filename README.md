#  Temperature Converter

A sleek and responsive Temperature Converter web application built using HTML, CSS, and JavaScript. This project allows users to quickly convert temperature values between Celsius, Fahrenheit, and Kelvin with real-time calculations and conversion history tracking.

##  Features

* Convert temperatures between:

  * Celsius (°C)
  * Fahrenheit (°F)
  * Kelvin (K)
* Instant and accurate conversions
* Clean and modern user interface
* Responsive design for desktop and mobile devices
* Conversion history tracking
* Input validation for better user experience
* Lightweight and fast performance

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
